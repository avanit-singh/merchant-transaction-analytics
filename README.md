## Overview
Analytics solution for payment gateway (PG) and merchant
transaction data. Tracks success rates, settlement TAT, MDR
performance, and payment failure patterns across merchant portfolios.

---

## Key Analyses
1. Transaction Success Rate & Failure Pattern Analysis
2. Settlement TAT (Turnaround Time) per Merchant
3. MDR (Merchant Discount Rate) Revenue Tracking
4. Merchant Performance Benchmarking
5. Active Merchant Ratio
6. Category-wise Payment Volume Analysis
